📘 ИНСТРУКТАЖ ПО ЗАПУСКУ И РАБОТЕ С ПРОЕКТОМ

1. Назначение проекта
Данный проект представляет собой программное средство для работы с расписанием, включающее:
загрузку данных (например, из Excel/CSV), обработку и анализ расписания,
визуализацию через веб-интерфейс (Dash), возможность оптимизации или проверки расписания.

2. Требования к системе
Перед запуском необходимо установить: Python 3.9+
pip (менеджер пакетов)
(рекомендуется) virtualenv или venv

3. Установка и подготовка

3.1. Распаковка проекта: Распаковать архив проекта в удобную папку.

3.2. Переход в директорию проекта: cd schedule_optimizer

3.3. Создание виртуального окружения (рекомендуется): python -m venv venv

Активация: Windows: venv\Scripts\activate
Linux / Mac:
source venv/bin/activate

3.4. Установка зависимостей:
pip install -r requirements.txt

Если файла нет: pip install dash pandas numpy plotly openpyxl

4. Запуск проекта
Основной запуск осуществляется через главный файл для выбора значения коэффициента:
python main.py

Для ввода значения коэффициента генератора данных с сохранением пропорций

Работа с приложением: 
Загрузка данных: 
Используются файлы формата при парсинге: 
.xlsx или .csv

Данные могут находиться в папке:
Д.1. Запуск системы

Убедитесь, что установлены все зависимости, перечисленные в pyproject.toml (или requirements.txt).

Для запуска веб-сервера выполните главный скрипт run.py из корневой директории проекта.

Дождитесь сообщения в консоли о старте Uvicorn-сервера.

Система автоматически откроет веб-интерфейс в браузере по умолчанию (обычно http://localhost:8000).

Д.2. Работа с пользовательским интерфейсом

Интерфейс системы представляет собой одностраничное приложение, 
разделенное на логические блоки.

1. Загрузка исходных данных
Если у вас есть готовые файлы, 
нажмите кнопку «Выберите файл» в областях 
«Ведомость занятий» и «Список аудиторий» 
и загрузите подготовленные .xlsx-файлы соответствующего формата.

Если файлы отсутствуют, система автоматически определит это 
и предложит сгенерировать демонстрационный набор данных. 
Это будет указано в интерфейсе.
2. Выбор алгоритма оптимизации
В блоке «Выбор алгоритма оптимизации» представлены 4 варианта. 
3. 
По умолчанию выбран «Комбинированный», который применяет все 3 алгоритма последовательно 

(Генетический → Имитация отжига → Жадный) для поиска наилучшего решения.

3. Запуск и анализ результатов
Нажмите кнопку «ЗАПУСТИТЬ ОПТИМИЗАЦИЮ РАСПИСАНИЯ». 
Во время работы кнопка станет неактивной.
Дождитесь завершения процесса. 
Время зависит от объема данных и выбранного алгоритма.
После завершения в правом блоке «Результаты оптимизации» отобразится ключевая информация:

Значение целевой функции F(X): 
Это главная метрика качества. 
Чем число меньше, тем лучше составлено расписание (меньше конфликтов и штрафов).
Детализация: 
Ниже будет выведено время работы алгоритма и количество нарушений каждого типа, таких как:
«Конфликты преподавателей» (накладки в расписании одного преподавателя).
«Конфликты групп» (накладки в расписании одной группы).
«Несоответствие аудиторий» (например, использование маленькой аудитории для большой группы).
«Штраф за окна» и «Неравномерность нагрузки».
4. Просмотр сгенерированного расписания
Пролистайте страницу вниз до динамически сформированной таблицы. 
Расписание сгруппировано по дням недели.
Ячейки таблицы: 
В каждой заполненной ячейке указаны: 
название дисциплины, 
учебная группа, 
ФИО преподавателя, 
номер аудитории 
и тип занятия (Лекция/Семинар).
Фильтрация: 
Вы можете просматривать расписание для конкретных сущностей, 
используя фильтры над таблицей 
(например, выбрать конкретную группу или преподавателя), 
если они предусмотрены в выпадающих списках.

Д.3. Экспорт и сохранение результатов
Для удобства дальнейшей работы система позволяет выгрузить полученное расписание.

Воспользуйтесь кнопкой «Экспорт в Excel» (или аналогичной) над таблицей, 

чтобы сохранить файл для печати, утверждения или распространения.

Сгенерированное расписание автоматически сохраняется во внутренней базе данных системы 
и будет доступно при следующих сеансах работы.
